# Enterprise-RAG

Production-grade Retrieval-Augmented Generation (RAG) framework designed for enterprise document intelligence.

## Features

- Layout-aware PDF parsing
- Semantic chunking
- Qdrant Vector Database
- Multi-tenant architecture
- Streamlit-based UI
- Docker deployment
- Environment-based configuration
